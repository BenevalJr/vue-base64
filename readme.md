# Vue Base64
An Example of handling base64 images in vue.js and Express Server. You can find a detailed walk through [here](https://medium.com/@martinsOnuoha/how-to-upload-base64-images-in-vue-nodejs-4e89635daebc)

## Directory Structure

```
.
|-- Index.vue
|-- assets
|-- package-lock.json
|-- package.json
|-- readme.md
`-- server
   |-- app.js
   |-- public

```
